## GitHub Repository Link:

## Project Description:

This project is developed as part of the VertexMind Frontend Internship Mission 2.
The main goal of this task was to add JavaScript functionality and interactivity to a frontend website.

## Features implemented:
- Contact form with input validation
- Dark mode toggle using JavaScript and localStorage
- API integration using Fetch API
- Dynamic rendering of API data
- Loading and error handling states

## Technologies Used:

- HTML5
- CSS3
- JavaScript (ES6)
- DOM Manipulation
- Fetch API
- LocalStorage
- VS Code
- Git & GitHub

 ## Challenges Faced:

- Adding JavaScript validation for form inputs
- Implementing dark mode functionality and saving user preference
- Fetching and displaying data from an external API
- Handling loading and error states properly
- Managing Git workflow and uploading updates to GitHub

## Outcome:

Successfully created an interactive frontend webpage with JavaScript features and uploaded the completed Mission 2 project to GitHub.
